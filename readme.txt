# Directorio "docs/"

El directorio **`docs/`** contiene la **documentación complementaria** del diseño de la base de datos de **eBORO**.
Su propósito es centralizar información que facilite la comprensión, mantenimiento y evolución del modelo de datos.

## Contenido

En este directorio se incluyen:

* **Diccionario de datos**:
  Define los atributos de cada tabla (nombre, descripción de la tabla, columnas, atributos, tip de dato, llave primaria, llave foranea, valor por dafault, constrains).

* **Diagramas de apoyo**:
  Diagramas ER, vistas lógicas exportadas desde ER-Studio.

## Objetivo

Este directorio busca servir como **guía de referencia** para desarrolladores y administradores de base de datos que interactúan con el modelo, asegurando:

* Comprensión uniforme del diseño.
* Facilitar auditorías y revisiones técnicas.
* Simplificar la capacitación de nuevos integrantes del equipo.
